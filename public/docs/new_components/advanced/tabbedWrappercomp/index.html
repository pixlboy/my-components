Pop Up Tabbed Table Component Demo
Explanation of component
  1. tabbed wrapper component which can have 'n' no.of tabs
  2. can pass pure html or config div to display different content for each tabs
  3. close button to the right to close the whole tabbed wrapper component
  4. background color as per active tab
Usage of Component
  1. step1: Create tabtitles how many ever tabs needed: 
    					var titles = ['tab0','allopen', ''allapproved]
  2. step2: Render tab titles 
    					var onClose = function(){
						ReactDOM.unmountComponentAtNode(document.getElementById('tabbedwrapper-popuptable'));
					}

					var onTabClicked = function(tabindex){
						JnprCL.TabbedWrapperStore('appId_1').setContent('html text should display ', tabindex);
					}

					ReactDOM.render(React.createElement(TabbedWrapperComp, {
					    appId: "appId_1",
						tabTitles: tabTitles ,
						onTabClicked: this.onTabClicked.bind(this),
						onClose: this.onClose.bind(this),

					}), document.getElementById('tabbedwrapper-popuptable'));
  3. You need to pass titles you create at step1 when creating the component
  4. you need to pass html or config into setContent like in step2 with tabindex
  5. You need to add onClose() handler to handle closing function of the table.
