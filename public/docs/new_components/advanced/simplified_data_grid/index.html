Simplified Data Grid
Explanation

Goal of Component:

  1. Simplified data grid, without the complicated functions as data table
  2. Ideal for pop up list of the erros, reasons etc.
  3. No sorting, filtering provided, based on pure HTML table

Usage of Component:

  • Step 1: create configuration: 
    						var config = {
							title: "Grid Title",
							columns: ['column1', 'column2', 'column3'],
							rows: [
								['row1_col1', 'row1_col2','row1_col3'],
								['row2_col1', 'row2_col2','row2_col3'],
								['row3_col1', 'row3_col2','row3_col3'],
								['row4_col1', 'row4_col2','row4_col3'],
								['row1_col1', 'row1_col2','row1_col3'],
								['row2_col1', 'row2_col2','row2_col3'],
								['row3_col1', 'row3_col2','row3_col3'],
								['row4_col1', 'row4_col2','row4_col3'],
								['row1_col1', 'row1_col2','row1_col3'],
								['row2_col1', 'row2_col2','row2_col3'],
								['row3_col1', 'row3_col2','row3_col3'],
								['row4_col1', 'row4_col2','row4_col3']
							]
						}
  • Step 2: Render Grid: 
    						var onClose = function(){
							ReactDOM.unmountComponentAtNode(document.getElementById('simplifiedDataGrid'));
						}

						var SimplifiedDataGrid = JnprCL.SimplifiedDataGrid;
						ReactDOM.render(React.createElement(SimplifiedDataGrid, {
							onClose: this.onClose.bind(this),
							config: config
						}), document.getElementById('simplifiedDataGrid'));
    1. You need to pass configruation created at step 1 when creating the component
    2. You need to add onClose() handler to handle closing function of the table.
