Export Component Document
Explanation

Goal of Component:

  1. Provide easy way to export customer data
  2. Only in-app export, do not use for global export, where, you need to call backend API to export
  3. Only CSV format, no excelsheet support is provided yet.

Usage of Component: 

				var ExportComponent = JnprCL.ExportComponent;
				ReactDOM.render(React.createElement(ExportComponent, {
					titles: ['column1', 'column2', 'column3'],
					dataList: [
					['row1_col1', 'row1_col2','row1_col3'],
					['row2_col1', 'row2_col2','row2_col3']
					],
					exportFileName: 'myFile.csv'
				}), document.getElementById('output'));
  1. titles: list of column titles for csv file
  2. dataList: list of rows data, and each row data is another list of cell data

Use of service directly In case you do not want to use this icon to download, you can use service directly. Please follow below: 

					var service = new JnprCL.ExportService(
						['column1', 'column2', 'column3'],
						[
							['row1_col1', 'row1_col2','row1_col3'],
							['row2_col1', 'row2_col2','row2_col3']
						]
					);
					service.download(true);
