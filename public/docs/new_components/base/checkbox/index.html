Checkbox Document
Explanation

Goal of Component: Simplify the usage of checkbox component

Usage of Component: 

				var CheckBox = JnprCL.CheckBox;
				var onChange = function(checked){
					document.getElementById('result').innerHTML = "checked result: " + checked;
				};
				ReactDOM.render(React.createElement(CheckBox, {
					label: "Select",
					default: true,
					onChange: this.onChange.bind(this)
				}), document.getElementById('output'));
  1. label (optional): label for the checkbox
  2. default (optional): true|false, false by default. Initial checkBox status
  3. onChange: callback handler when you change the checkbox select status
