CloseButton Document
Explanation

Goal of Component: We have a lot place to close a popup, and the requirements are:

  1. By clicking the close button
  2. By clicking anywhere out of the wrapper application / popup component
  3. By click ESC key to close

Usage of Component: 

				var CloseButton = JnprCL.CloseButton;
				var close = function(){
					ReactDOM.unmountComponentAtNode(document.getElementById('output'));
				};
				ReactDOM.render(React.createElement(CloseButton, {
					wrapperClassName: 'output',
					onClick: this.close.bind(this)
				}), document.getElementById('output'));
  1. wrapperClassName (mandatory): this is pointing to the wrapper appliacation className, and it should be outermost className to be used to locate which part to destroy
  2. onClick (mandatory): callback function used to close the component. Here, you can use any solution to unmount the component.
